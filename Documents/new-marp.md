# Overview

- Investigating the feasibility of blood transmutation using machine learning.
- Analyze blood group antigens, genetic markers, and biochemical properties.
- Aim: Enable safe alteration of blood types, improving transfusion safety.

- **Reference**: [Toward Universal Donor Blood](https://www.researchgate.net/publication/337684943_Toward_universal_donor_blood_Enzymatic_conversion_of_A_and_B_to_O_type)

---

# Data Collection & Preprocessing

- **Blood Data:**
  - Blood Group, Rh Factor, Hemoglobin, White Blood Cell Count, etc.
- **Chemical Components Data:**
  - Molecular Formula, Concentration, Reaction Types, etc.
- **Conversion Methods Data:**
  - Techniques, Effectiveness, Required Chemicals.
- **Experimental & Clinical Data:**
  - Lab Results, Clinical Trials.

---

# Machine Learning Models

- **Base Models:**
  - Random Forest
  - XGBoost
- **Meta-Model:**
  - Neural Network
- **Ensemble Learning:** Combining the models for better predictions.

---

# Implementation Steps

1. Train base models (Random Forest, XGBoost).
2. Generate predictions and combine them.
3. Train a neural network on combined predictions.
4. Evaluate model performance on test data.
